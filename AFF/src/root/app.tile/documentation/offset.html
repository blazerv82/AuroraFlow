Offset
o-[Size]
REQUIRED

Size - How far a container/object is from the left of the parent container/object

Offset numbering is the same as Viewport numbering. Using a page broken up into 12 sections, a container set to the size of vl-6 using an offset of o-6 will start halfway, and end at the rightmost edge of the parent element.
