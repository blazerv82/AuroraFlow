Viewport
v[Viewport]-[Width]
REQUIRED

Viewport - What minimum size viewport to use
Width - How wide the container or object is

This is used to set the size of a flex-container at a given viewport size.

SIZING
Small [VS]
Under 525px
Medium [VM]
525px to 950px
Large [VL]
950px to 1366px
Xtra Large [VXL]
Over 1366px
