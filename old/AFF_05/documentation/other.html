Other

class="border-none" Removes a border, if something inherently supports it

class="border" Adds a border of 1px, can be styled via color classes

class="border-vl" Adds a border of 5px, can be styled via color classes

class="border-round-none" Removes border radius

class="border-round" Adds a border radius of 10px

class="border-round-vl" Adds a border radius of 25px

class="shadow" Adds a drop shadow of 15px
