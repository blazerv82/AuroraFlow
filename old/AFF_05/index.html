AuroraFlow: Dashboard

Current version is V5: Empirical

Last major updates made in September 2019

AuroraFlow uses Normalize.CSS to reset several parts of a webpage, to ensure consitent styling and a baseline for the custom styling AuroraFlow provides. That said, div and span tags are the only tag fully supported with styling/sizing. p and pre tags are supported for the text-container element. Use other tags at your own risk!
