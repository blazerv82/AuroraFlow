Display

class="d-block" Displays an element as block

class="d-inline-block" Displays an element as inline-block

class="d-flex-row" Sets the flex display of a container to row

class="d-flex-column" Sets the flex display of a container to column

class="h-auto" Sets the height to the full size of its parent container

class="w-auto" Sets the width to the full size of its parent container
