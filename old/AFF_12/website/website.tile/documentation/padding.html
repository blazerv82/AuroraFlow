Padding
.p [X/Y] -[SIZE] [-REMOVE]
p[X/Y]-[Size][-Remove]
REQUIRED

X/Y - Axis of padding
Size - How much padding to apply

Padding is equally applied to both sides of an element on the axis specified using EM units, unless otherwise stated using a postfix parameter.

OPTIONAL

Remove - Which side to remove padding

Options for the [-Remove] parameter: l[eft] r[ight] t[op] b[ottom]

px-small
px-medium
px-large
px-xlarge
py-small
py-medium
py-large
py-xlarge
px-none
px-none-l
px-none-r
py-none
py-none-t
py-none-b

As of Flowing Forest [9.0], it is recommended to remove padding ONLY on one side of an element by adding the prerequisite padding to the axis needed, then removing the unwanted side using both style parameters. This may be changed at a later time.
