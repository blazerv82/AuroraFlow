Viewport chevron_right Explained
Viewport - What minimum size viewport to use
Size - How wide the container or object is
This is used to set the size of a .flex-container at a given viewport size. You can set multiple options for content to display differently at some or all viewport sizes
Viewport chevron_right Types
s 0px ~ 480px - Mobile Tall
m 480px ~ 925px - Tablet Tall, Mobile Wide
l 925px ~ 1366px - Tablet Wide, PC Small
xl 1366px ~ 1920px - PC Medium
2xl 1920px ~ Larger - PC Large
