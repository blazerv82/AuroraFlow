Round
round-[SIZE][-SIDE]
ROUND chevron_right SIZE
none Removes any preset round
small Set the round to a small size, based on EM units (0.25em)
medium Set the round to a medium size, based on EM units (0.5em)
large Set the round to a large size, based on EM units (1em)
ROUND chevron_right SIDE
l Round only on the left side of the element
r Round only on the right side of the element
t Round only on the top side of the element
b Round only on the bottom side of the element
x Round the X axis of the element, shorthand for L+R
y Round the Y axis of the element, shorthand for T+B

As of Cosmic Voltage - [10.0], it is not possible to combine seperate sides, such as L+T, due to each style overwriting the other. A more refined rounding system which allows mix and match may come at a later date.
