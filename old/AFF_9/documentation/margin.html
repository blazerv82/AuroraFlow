Margin
m[X/Y]-[Size][-Remove]
REQUIRED

X/Y - Axis of margin
Size - How much margin to apply

Margin is equally applied to both sides of an element on the axis specified using EM units, unless otherwise stated using a postfix parameter.

OPTIONAL

Remove - Which side to remove padding

Options for the [-Remove] parameter: l[eft] r[ight] t[op] b[ottom]

mx-small
mx-medium
mx-large
mx-xlarge
my-small
my-medium
my-large
my-xlarge
mx-none
mx-none-l
mx-none-r
my-none
my-none-t
my-none-b

As of Flowing Forest [9.0], it is recommended to remove margin on ONLY one side of an element by adding the prerequisite margin to the axis needed, then removing the unwanted side using both style parameters. This may be changed at a later time.
