Display
BASIC

d-none Removes an object from displaying on the page
d-block Displays an object as block
d-inline-block Displays an object as inline-block

FLEXBOX // BASIC

d-flex-column Sets the display of a container to flex, with a column layout
d-flex-row Sets the display of a container to flex, with a row layout

FLEXBOX // ITEMS

d-flex-center-all Sets the flex items of a container to display in the center
d-flex-justify-all Sets all flex item of a container to display evenly spaced within a container
d-flex-center-s Sets the flex items of a container to display in the center, but spaced evenly
d-flex-center-self Sets one flex item of a container to display in the center

POSITIONING

d-pos-absolute Sets a container or object to display in an absolute position within the webpage
d-pos-fixed Sets a container or object to display in a fixed position within the window
d-pos-sticky Sets a container or object to display in a sticky position within the window. The object will act as normal until no longer on the webpage fully, then will 'stick' to that part of the window

OTHER

h-auto Sets the height to the full size of its parent container
w-auto Sets the width to the full size of its parent container
