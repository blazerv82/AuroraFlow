CONTAINERS
class="container"
Defines a container for holding all HTML content.
In VXL viewports, any nested content container will have a small amount of padding from one another
class="content-container"
Defines a container for holding all HTML content, along with a small amount of padding. Can be nested
class="image-container"
Defines an image container for holding images. Displays as flex for centering images, but can be overridden.
class="code-container"
Defines a code container for showing snippets of code. WILL render code!
class="header-container"
Defines a header container for alerting someone to an important or new section
class="nav-container"
Defines a navigation section for the page
