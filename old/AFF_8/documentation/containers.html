Containers
container

Defines a basic top-level container for holding all HTML content

spaced-container

Defines a basic top-level container for holding all HTML content along with padding on both sides. Will stay at a fixed width in all viewports except for VS, where it will take up the full space available

row

Defines a row container for defining rows of HTML content, used in conjunction with content containers to create a row/column relationship. Anything within a row will be set to display: flex; with automatic wrapping

content-container

Defines a row container for defining rows of HTML content, used in conjunction with row containers to create a row/column relationship

text-container

Defines a text container for showing lines of text content. Adds a small amount of extra X padding for readability

image-container

Defines an image container for holding images. Displays as flex for centering images, but can be overridden

code-container

Defines a code container for showing snippets of code, in a header style

inline-code-container

Defines an inline code container for showing snippets of code

header-container

Defines a header container for alerting someone to an important or new section

nav-container

Defines a navigation section for the page

alert-container

Defines an alert container, usually used to alert a user to something
