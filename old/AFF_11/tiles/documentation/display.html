Display
d-[OPTIONS]
BASIC
d-none Removes an element from displaying on the page
d-block Displays an element as block
d-inline-block Displays an element as inline-block
d-absolute Displays an element within an absolute position within the webpage
d-fixed Displays an element in a fixed position within the window
d-sticky Displays an element within a sticky position within the window. The element will act as normal until no longer on the webpage fully, then will 'stick' to that part of the window. Useful for navigation bars or footers.
d-float-r Forces an element to be at the rightmost side of its parent element
d-float-l Forces an element to be at the leftmost side of its parent element
FLEXBOX
d-flex-column Sets the display of an element to flex, with a column layout
d-flex-row Sets the display of an element to flex, with a row layout
d-flex-center-all Sets the flex items of an element to display in the center of the container or object
d-flex-justify-all Sets all flex item of an element to display evenly spaced within the container or object
d-flex-center-self Sets one flex item of an element to display in the center within the container or object
d-flex-center Sets the flex items of an element to display in the center and justified, of the container or object. Shorthand for d-flex-center-all and d-flex-justify-all combined
OTHER
h-auto Sets the height an element to the full size of its parent
w-auto Sets the width of an element to the full size of its parent
