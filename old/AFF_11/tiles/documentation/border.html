Border
border-[SIZE][-SIDE]
border-[SPECIAL]
BORDER chevron_right SIZE
none Removes any preset border
thin Set the border to a small size, based on PX units (1px)
small Set the border to a small size, based on EM units (0.25em)
medium Set the border to a medium size, based on EM units (0.5em)
large Set the border to a large size, based on EM units (1em)
BORDER chevron_right SIDE
l Displays the border only on the left side of the element
r Displays the border only on the right side of the element
t Displays the border only on the top side of the element
b Displays the border only on the bottom side of the element
x Displays the border on the X axis of the element, shorthand for L+R
y Displays the border on the Y axis of the element, shorthand for T+B

As of Cosmic Voltage - [10.0], it is not possible to combine seperate sides, such as L+T, due to each style overwriting the other. A more refined border system which allows mix and match may come at a later date.
