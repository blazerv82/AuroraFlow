Containers and Objects
CONTAINERS chevron_right TOP LEVEL ELEMENTS
container Defines a basic top-level container for holding all HTML content
spaced-container Defines a basic top-level container for holding all HTML content along with padding on both sides.
Will stay at a fixed width in all viewports except for VS, where it will take up the full space available
flex-container Defines a flexbox container to create a responsive looking webpage
nav-container Defines a navigation section for the page
footer-container Defines a footer section for the page
OBJECTS chevron_right SUB-LEVEL ELEMENTS
section-object Defines a container of HTML content.
spaced-object Defines a container of HTML content for use within flex-container or flex-container-full that automatically adds a small margin to every element
text-object Defines a container of HTML content that will display text.
header-object Defines a container of HTML content that will display something important in large text, like an h1 tag would do.
message-object Defines a container of HTML content that will display something important
code-object Defines a container of HTML content that will display some code with monospaced font
inline-code-object Defines a container of HTML content that will display some code, but within the content itself, rather than a large container
