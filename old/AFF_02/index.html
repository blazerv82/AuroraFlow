AURORA FLOW FRAMEWORKS

Current version is V4: Normalize'd Edition

Updated last on May 14 2019

AuroraFlow uses Normalize.CSS to reset several parts of a webpage, to ensure consitent styling and a baseline for the custom styling AuroraFlow provides. That said, div tags are the only tag fully supported with styling/sizing. p tags are supported for the text-container element. Use other tags at your own risk!
